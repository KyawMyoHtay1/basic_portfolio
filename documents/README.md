# CV PDF

Place your résumé here:

**File name (required):** `Kyaw_Myo_Htay_CV.pdf`

Full path:

```
portfolio/documents/Kyaw_Myo_Htay_CV.pdf
```

Then redeploy:

```powershell
powershell -ExecutionPolicy Bypass -File portfolio\deploy.ps1
```

The site will show the PDF on the **CV** page and enable **Download CV** buttons.
